# 🚀 快速开始指南

## 第一步：准备工作

### 1. 注册微信小程序账号

访问：https://mp.weixin.qq.com/

- 点击"立即注册"
- 选择"小程序"类型
- 填写邮箱、密码等信息
- 完成邮箱激活

**注意**：个人主体即可，无需营业执照

---

### 2. 获取 AppID

登录小程序后台：

- 进入 **开发 → 开发管理 → 开发设置**
- 复制 **开发者 ID** 中的 `AppID(小程序 ID)`
- 格式类似：`wx1234567890abcdef`

---

### 3. 安装微信开发者工具

下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

- 选择对应系统版本（Mac/Windows）
- 安装后使用微信扫码登录

---

### 4. 开通云开发

在微信开发者工具中：

1. 导入项目（下一步）
2. 点击工具栏 **云开发** 按钮
3. 开通云开发（免费）
4. 记录 **环境 ID**（格式类似：`cloud1-xxx`）

---

## 第二步：导入项目

### 1. 克隆代码

```bash
cd ~/.openclaw/workspace/projects
git clone https://github.com/YOUR_USERNAME/couple-trip-map.git
```

或者手动下载 ZIP 解压到本地。

---

### 2. 导入开发者工具

1. 打开微信开发者工具
2. 点击 **+** 导入项目
3. 项目目录：选择 `couple-trip-map` 文件夹
4. AppID：填入你的 AppID
5. 点击 **导入**

---

### 3. 修改配置

#### 修改 `miniprogram/app.js`

```javascript
wx.cloud.init({
  env: 'your-env-id',  // 改为你的云开发环境 ID
  traceUser: true,
})
```

#### 修改 `project.config.json`

```json
{
  "appid": "wx YOUR_APP_ID",  // 改为你的 AppID
  "projectname": "couple-trip-map"
}
```

---

## 第三步：初始化云开发

### 1. 创建数据库集合

在云开发控制台：

1. 点击 **数据库**
2. 点击 **+** 创建集合
3. 创建以下集合：
   - `users`
   - `couples`
   - `trips`
   - `locations`
   - `photos`

---

### 2. 设置权限

每个集合的权限设置为：

```
自定义规则：
{
  "read": "auth.openid == doc._openid",
  "write": "auth.openid == doc._openid"
}
```

**注意**：情侣共享数据需通过云函数代理访问

---

### 3. 上传云函数

在云开发控制台：

1. 点击 **云函数**
2. 点击 **+** 导入云函数
3. 选择 `cloudfunctions/login` 文件夹
4. 等待上传完成
5. 右键云函数 → **部署**

---

### 4. 初始化地点数据

创建 `locations` 集合后，导入基础省份数据：

```javascript
// 在云开发控制台的数据库页面，点击"导入"
[
  { "_id": "beijing", "name": "北京市", "level": "province", "lat": 39.9042, "lng": 116.4074 },
  { "_id": "shanghai", "name": "上海市", "level": "province", "lat": 31.2304, "lng": 121.4737 },
  { "_id": "guangdong", "name": "广东省", "level": "province", "lat": 23.1291, "lng": 113.2644 },
  // ... 更多省份
]
```

## 第四步：运行调试

### 1. 编译预览

在开发者工具中：

- 点击 **编译** 按钮
- 查看模拟器效果
- 点击 **预览** → 手机扫码真机测试

---

### 2. 调试技巧

- **Console 面板**：查看日志输出
- **Storage 面板**：查看本地存储
- **AppData 面板**：查看数据状态
- **云开发控制台**：查看数据库和云函数

---

## 第五步：获取地图 Key

### 高德地图 Key

1. 访问：https://lbs.amap.com/
2. 注册/登录账号
3. 进入 **控制台 → 应用管理 → 我的应用**
4. 点击 **创建新应用**
5. 创建 Key：
   - 服务平台选择：**小程序**
   - 类型选择：**微信小程序**
6. 复制 Key 到代码中使用

---

## 常见问题

### Q1: 云开发环境 ID 在哪看？

云开发控制台 → 设置 → 环境 ID

### Q2: 真机预览报错？

- 检查 AppID 是否正确
- 检查云开发环境 ID 是否正确
- 确保手机已绑定为体验者（小程序后台 → 成员管理）

### Q3: 云函数调用失败？

- 检查云函数是否已部署
- 检查云函数名称是否正确
- 查看云函数日志排查错误

### Q4: 图片上传失败？

- 检查云存储空间是否充足
- 检查网络权限
- 查看云存储文件列表确认是否上传成功

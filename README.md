# 💕 遇亭 (YuTing) - 情侣旅行地图

> **异地情侣的旅行记忆地图 —— 记录俞和婷的每一次相聚**
> 
> “遇亭”取自“俞”与“婷”的邂逅，也是一段跨越距离、记录温暖的旅程。

---

## 🎯 项目定位

遇亭是一款专为异地情侣设计的旅行记录应用。它不仅仅是一个打卡工具，更是承载共同回忆的数字空间。通过地图可视化、多层级导航和时间线相册，让每一次相聚都有迹可循。

## ✨ 核心功能

- 🗺️ **足迹地图**：中国地图可视化展示，实时点亮已探索省份
- 🔍 **多级下钻**：支持 `省份 → 城市 → 景点` 的深度探索与打卡
- 📸 **共同相册**：每个地点绑定多张照片，按时间线记录甜蜜瞬间
- 💑 **情侣绑定**：双人账户关联，支持共同编辑与实时数据同步
- 📊 **数据中心**：去过省份/城市统计、旅行天数计算、足迹覆盖率统计
- 🎨 **精美海报**：一键生成定制化旅行地图海报，分享你们的爱情地图

## 🛠️ 技术架构

遇亭采用全栈 Serverless 架构，确保存储可靠、响应迅速且运维成本低。

| 层级 | 技术选型 | 备注 |
| :--- | :--- | :--- |
| **前端** | 微信小程序原生 / Taro / Next.js | 多端适配支持 |
| **后端** | 微信云开发 / Supabase | 强大的 Serverless 后端 |
| **地图** | 高德地图 JS API | 高精度的地图可视化服务 |
| **存储** | 云开发云存储 / Supabase Storage | 安全的照片与资源存储 |
| **数据库** | 云开发数据库 / PostgreSQL | 高性能 JSON/关系型存储 |

## 📁 项目结构

```text
couple-trip-map/
```

## 🚀 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/sy0310/couple-trip-map.git
   ```

2. **环境配置**
   - 微信开发者工具导入 `miniprogram` 目录。
   - 在 `app.js` 中配置您的云开发环境 ID。
   - `yuting` 目录运行 `npm install && npm run dev` 启动 Web 端。

3. **初始化数据库**
   - 参考 `docs/database.md` 在云开发控制台创建相应的集合。

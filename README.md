# 💕 情侣旅行地图 (Couple Trip Map)

> 异地恋情侣的旅行记忆地图 —— 点亮一起去过的城市，珍藏每一段旅程

## 🎯 项目定位

专为异地情侣设计的旅行记录小程序，通过地图可视化共同旅行记忆，支持省→市→景点多层级打卡，照片 + 时间 + 地点绑定。

## ✨ 核心功能

- **地图总览**：中国地图展示去过的省份，点亮效果
- **多层级导航**：省 → 市 → 景点 逐级下钻
- **照片墙**：每个地点绑定多张照片，时间线展示
- **情侣绑定**：双人账户关联，共同编辑
- **数据统计**：去过城市数、旅行天数、里程统计
- **分享功能**：生成旅行地图海报，可选公开/私密

## 🛠️ 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | 微信小程序 (原生) |

## 🚀 快速开始

### 1. 环境准备

- 微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 小程序账号：https://mp.weixin.qq.com/
- 开通云开发：开发者工具 → 云开发 → 开通

### 2. 配置项目

```bash
# 克隆项目
git clone https://github.com/YOUR_USERNAME/couple-trip-map.git

# 导入微信开发者工具
# 文件 → 导入项目 → 选择项目目录
```

### 3. 修改配置

编辑 `miniprogram/app.js`，替换你的云开发环境 ID：

```javascript
wx.cloud.init({
  env: 'your-env-id'  // 替换为你的云开发环境 ID
})
```

### 4. 运行

- 微信开发者工具 → 编译 → 预览
- 真机调试：工具 → 预览 → 扫码

## 📋 已完成功能

### P0 优先级功能
- **个人中心页面** (`pages/profile/`) - 情侣绑定/解绑功能
- **情侣关系云函数** (`cloudfunctions/couple/`) - 绑定码生成、加入/创建情侣关系
